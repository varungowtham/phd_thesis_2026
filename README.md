## Template instructions

- All instructions are included in the first chapter of the "dbta_Styleguide"
- Please read it carefully!

## Recommended TeXLive versions
- earlier than v2.3.4: please consult the change log for the appropriate TeXLive version
- from v2.3.4: TexLive 2020
- from v3.2: TeXLive 2021
- from v3.3.1: TeXLive 2022
- from v3.4.0: TexLive 2023

## How to clone

- install git on your computer 
- open a console window
- cd to the directory where you want the folder to be
- type "git clone https://git.tu-berlin.de/dbta/templates/Thesis_template.git" (without quotes)
- alternative: use the download button without having to have git installed
